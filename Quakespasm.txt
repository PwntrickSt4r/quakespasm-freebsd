QuakeSpasm v0.96.3 (FreeBSD Build)
==================================

Clean build from original source code.

Dependencies:
# pkg install gmake pkgconf sdl2 libvorbis libmad

How to run:
1. Download the binary from the Releases section.
2. Put the 'quakespasm' binary into your Quake folder (where the 'id1' directory is).
3. Run: ./quakespasm
